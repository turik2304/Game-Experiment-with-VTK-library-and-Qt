# Game-Experiment-with-VTK-library-and-Qt-
In this "game", you can:

-move (up-down, left-right buttons), 

-rotate the map - mouse, 

-shoot (right Shift), 

-change the projectile speed ('v' - up speed, 'c' - down speed), 

-change the projectile range ('z' - range decreases, 'x' - range increases), 

-change the projectile size ('b' - size decreases, 'n' - size increases)
